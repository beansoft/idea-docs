File and Class Choosers List and Tree Controls

Editor Components

EditorTextField

相比

Swing JTextArea，

IntelliJ Platform编辑器组件具有很多优点:语法高亮支持，代码完成，代码折叠等等.

IntelliJ Platform 编辑器通常显示在编辑器选项卡中，但它们也可以嵌入到对话框或工具窗口中.

这是由

EditorTextField零件.

创建时

EditorTextField，

您可以指定以下属性:

  • 解析文本字段中文本的文件类型;

  • 文本字段是只读还是可编辑;

  • 文本字段是单行还是多行.

一个常见的用例

EditorTextField

输入Java类或包的名称.

这可以通过以下步骤完成:

  • 使用     JavaCodeFragmentFactory.getInstance().createReferenceCodeFragment()     创建表示类或包名称的代码片段;

  • 打电话     PsiDocumentManager.getInstance().getDocument()     获取与代码片段对应的文档;

  • 将退回的文件传递给     EditorTextField     构造函数或其setDocument()方法.
